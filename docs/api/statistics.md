# Statistics API

This page provides API references for MinexPy's statistics functionality.

## Statistical Analysis

::: minexpy.stats
    options:
      show_root_heading: true
      show_root_toc_entry: true
      show_source: true
      heading_level: 2

## Correlation Analysis

::: minexpy.correlation
    options:
      show_root_heading: true
      show_root_toc_entry: true
      show_source: true
      heading_level: 2

## Statistical Visualization

::: minexpy.statviz
    options:
      show_root_heading: true
      show_root_toc_entry: true
      show_source: true
      heading_level: 2
